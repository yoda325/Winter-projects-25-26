# Meta-Learning for Wireless Localization

## Project Overview
We built a **3D indoor wireless localization** system leveraging meta-learning paradigms. The model predicts a user's (x, y, z) position by relying solely on the Received Signal Strength (RSSI) measured from 4 surrounding Wi-Fi Access Points. We successfully implemented **both MAML (Model-Agnostic Meta-Learning) and Reptile** to allow our neural network to fast-adapt to entirely new room environments utilizing only 10 labelled coordinate samples.
